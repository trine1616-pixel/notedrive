# QA 실행 로그 (QA Log)

이 문서는 실기기/환경별 테스트 결과를 기록하는 운영 로그입니다.

## 기록 규칙
- 날짜: `YYYY-MM-DD`
- 환경: `macOS / Windows / Android / Browser`
- 빌드/실행 방식: `dev` 또는 `stable(launcher)`
- 결과: `PASS / FAIL / BLOCKED`
