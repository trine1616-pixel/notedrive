# 역할 맵 (Role Map)

## 활성 역할 (권장 6개)
- ProductOwner
- PlatformEngineer
- FrontendUXEngineer
- AIEngineer
- SecurityPrivacyEngineer
- ReleaseOrchestrator

## 권장 협업 구조
1. ProductOwner가 우선순위/DoD를 확정합니다.
2. Platform/Frontend/AI/Security가 병렬로 구현합니다.
3. ReleaseOrchestrator가 검증, 브랜치, PR, 머지를 관리합니다.
4. 문서/체크리스트를 동기화하고 릴리스 결정을 내립니다.
