# 세션 시작 지침 (Session Start)

새로운 세션이 시작되었습니다. 다음 단계를 수행하여 컨텍스트를 복원하십시오.

1. `docs/handoff.md`를 읽고 마지막 작업 상태를 확인합니다.
2. `docs/project_context.md`와 `docs/architecture.md`를 로드하여 프로젝트의 목적과 설계를 파악합니다.
3. `docs/tasks.md`를 확인하여 현재 진행 중인 태스크를 파악합니다.
4. `docs/decisions.md`를 통해 기존의 기술적 결정을 존중합니다.
5. 사용자에게 현재 파악한 상태를 요약 보고하고 작업을 재개할 준비가 되었음을 알립니다.
