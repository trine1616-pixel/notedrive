# **App Name**: NoteDrive

## Core Features:

- Hierarchical Folder Management: Organize notes within nested folders for easy navigation.
- Markdown Editing Interface: Provide a user-friendly markdown editor with real-time preview.
- Hashtag System: Enable hashtagging within notes for categorization and search.
- Google Drive Integration: Automatically save and synchronize `.md` files with Google Drive.
- Note Search: Quickly find notes based on keywords, hashtags, or content.
- AI-Powered Note Summarization: Generate concise summaries of long notes using generative AI as a tool.

## Style Guidelines:

- Primary color: Light blue (#ADD8E6) to evoke a sense of calm and clarity.
- Background color: Off-white (#F5F5F5) for a clean and distraction-free writing environment.
- Accent color: Darker blue (#0077CC) for interactive elements and highlights, ensuring good contrast.
- Body and headline font: 'Inter' sans-serif for a clean and modern look.
- Use simple, line-based icons for folder and note actions.
- Intuitive sidebar for folder navigation and note list display.
